# Веб-интерфейс GANDM

React + TypeScript + Vite. Публичная часть разделена на поиск грузов,
транспорта, складов и таможенных услуг. После входа все участники попадают в
один кабинет, а разделы и действия определяются выбранными инструментами.

## Запуск

```bash
npm ci
npm run dev
```

Интерфейс откроется на `http://localhost:5173`. Backend должен работать на
`http://localhost:8080`; Vite проксирует запросы `/api`, поэтому отдельный CORS
для локальной разработки не нужен.

## Проверки

```bash
npm test
npm run lint
npm run build
npm audit
```

## Переменные окружения

Скопируйте `web/.env.example` в `web/.env`. Для китайской карты нужен
`VITE_AMAP_KEY`; без него интерфейс показывает понятное сообщение. Все точки,
отправляемые в API, преобразуются в WGS-84.

## Структура

- `src/api` — типизированные обращения к API;
- `src/auth` — сессия и безопасное обновление access-токена;
- `src/components` — общие элементы, карты и компоновка;
- `src/i18n` — русская, английская и китайская локализации;
- `src/pages` — публичные страницы, общий кабинет и админка;
- `src/utils/toolSections.ts` — единая группировка инструментов по услугам;
- `src/styles/global.css` — адаптивные стили.

Access-токен хранится только в памяти, refresh-токен — в `httpOnly` cookie.
В `localStorage` остаются только несекретные данные профиля.
